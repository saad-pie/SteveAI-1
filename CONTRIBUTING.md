# Contributing to SteveAI

We love help! To contribute to the Ultimate AI Orchestrator:
1. **Fork** the repository.
2. **Create a branch** for your new model integration (`git checkout -b feature/new-model`).
3. **Commit** your changes with clear messages.
4. **Push** and open a **Pull Request**.

### Code Style
Please follow the settings in our `.editorconfig` to keep the code clean for Saadpie & Ahmed.
